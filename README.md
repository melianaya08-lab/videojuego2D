## Videojuego 2D - ALIENS CAT

## Contexto

El videojuego presenta un escenario espacial donde diferentes gatos con apariencia alienígena se desplazan constantemente en distintas direcciones. Estos objetos interactúan entre sí mediante colisiones, generando efectos visuales que simulan choques en el espacio.

## Objetivo

El objetivo del jugador es eliminar la mayor cantidad de gatos alienígenas posibles mediante clics, incrementando así su puntaje.

## Justificación

Este proyecto se desarrolló con el fin de aplicar conceptos de programación en JavaScript utilizando Canvas 2D, incluyendo animación, control de eventos, uso de imágenes y detección de colisiones entre objetos.

## Operación

* Los gatos se mueven en múltiples direcciones con distintas velocidades
* El jugador utiliza una nave espacial (cursor) para hacer clic sobre los objetos
* Al hacer clic en un gato, este desaparece y aparece en otra posición aleatoria
* Las colisiones entre gatos generan un breve destello visual
* Se muestra un contador con el número de objetos eliminados

## Conclusión

El desarrollo del videojuego permitió comprender el manejo de animaciones en tiempo real, así como la interacción entre el usuario y los elementos gráficos en un entorno dinámico.
